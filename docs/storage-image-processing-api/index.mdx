# Storage Image Processing Api

Welcome to Storage Image Processing Api!

Optimize and transform images from your Cloud Storage buckets via a powerful HTTP API. Library Supports over 30 different image operations to optimize and manipulate your images.

## Getting started

Follow our guide on how to setup and process images [here](/getting-started).

## Interactive Example

<div class="flex flex-col">
  <img
    id="imgSrc"
    src="https://europe-west2-extensions-testing.cloudfunctions.net/ext-storage-image-processing-api-handler/process/input~type:create~width:200~height:100/flatten~background:black/text~value:Invertase/output~format:jpeg"
  />
  <div class="flex">
    <div>url:</div>
    <input
      type="text"
      value=""
      onChange={({ target }) => {
        var image = document.getElementById('imgSrc');
        image.src = target.value;
      }}
    />
  </div>
  <div class="flex">
    <label>Image text</label>
    <input
      type="text"
      onChange={({ target }) => {
        var image = document.getElementById('imgSrc');
        image.src = image.src.replace(
          /text~value:[a-zA-Z]*/,
          `text~value:${target.value}`,
        );
      }}
    />
  </div>
  <div class="flex">
    <label>Width (in pixels)</label>
    <input
      type="number"
      onClick={() => {
        var image = document.getElementById('imgSrc');
        image.src = image.src.replace(
          /width:[0-9]*/,
          `text~value:${target.value}`,
        );
      }}
    />
  </div>
</div>
