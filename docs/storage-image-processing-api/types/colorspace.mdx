### `Colorspace`

Set the output colorspace. By default output image will be web-friendly sRGB, with additional channels interpreted as alpha channels.

| Values    |
| --------- |
| `cmyk`    |
| `labq`    |
| `rgb`     |
| `cmc`     |
| `labs`    |
| `labs`    |
| `fourier` |
| `rgb16`   |
| `grey16`  |
| `matrix`  |
| `scrgb`   |
| `hsv`     |
